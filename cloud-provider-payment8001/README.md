# HTTP请求返回的状态码
| 分类  | 区间      | 分类描述                    |
|-----|---------|-------------------------|
| 1** | 100~199 | 信息，服务器收到请求，需要请求者继续执行操作  |
| 2** | 200~299 | 成功，操作被成功接收并处理           |
| 3** | 300~399 | 重定向，需要进一步的操作以完成请求       |
| 4** | 400~499 | 客户端错误，请求包含语法错误或无法完成请求   |
| 5** | 500~599 | 服务器错误，服务器在处理请求的过程中发生了错误 |
